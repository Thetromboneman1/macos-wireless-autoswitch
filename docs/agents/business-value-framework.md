# Business Value Framework

Date: 2026-06-23
Owner: AI Workforce Program

## Purpose

The business value framework measures whether operational agents reduce effort, improve decisions, find automation opportunities, and increase operational efficiency.

## Metrics

| Metric | Definition | Evidence |
|---|---|---|
| Time saved | Estimated manual hours avoided by an agent output | Before/after estimate, owner confirmation |
| Automation opportunities found | Count of validated candidates with owner and next action | Automation Discovery output |
| Architecture effort reduced | Review or drafting hours avoided for TAP Lite, ADR, or architecture updates | Architecture owner estimate |
| Reporting effort reduced | Hours avoided for monthly, quarterly, or executive reporting | Executive Communications run record |
| Operational efficiency | Reduced repeat work, faster RCA, fewer unresolved exceptions | Operational review, runbook, incident evidence |
| Governance readiness | Outputs passing evaluation without major rework | Evaluation score and findings |

## Reporting Cadence

| Cadence | Report |
|---|---|
| Weekly | Automation candidates found and next actions |
| Monthly | Time saved, reporting effort reduced, operational exceptions |
| Quarterly | Architecture effort reduced, ROI themes, executive impact |

## Validation

Business value claims require an owner, evidence source, calculation method, and review date. Estimates are acceptable when labeled as estimates and reviewed in the next reporting cycle.
